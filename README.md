# Web_lab_4

Фитнес-трекер.Веб-приложение для отслеживания фитнес-тренировок с визуализацией прогресса и детальной статистикой.

Возможности:
Дашборд с графиками - Визуализация прогресса по калориям и типам тренировок

Добавление тренировок - Интуитивная форма ввода с предпросмотром

Детальная статистика - Фильтрация по периодам и типам тренировок

История тренировок - Поиск, просмотр и управление всеми записями

Адаптивный дизайн - Оптимизирован для всех устройств

Локальное хранение - Данные сохраняются в браузере (localStorage)

Структура проекта

fitness-tracker/ 

├── index.html # Главная HTML страница 

├── style.css # Стили всего сайта 

├── js/ # Папка с JavaScript файлами 

├── data-manager.js # Управление данными (localStorage) │

├── charts.js # Графики и визуализация 

└── script.js # Основная логика приложения 

└── README.md # Документация проекта
